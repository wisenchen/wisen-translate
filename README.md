# wisen-translate

## 内嵌翻译网页
这个插件嵌入了有道，百度的翻译页面可以在vscode中直接打开
[gif]
## 使用
鼠标悬浮翻译
[gif]

### 快捷键

| 按键  | 说明                               |
| :---: | ---------------------------------- |
| alt+t | 打开一个输入框可输入中文翻译成英文 |
| alt+w | 翻译选择文字                       |



### 配置项

| 名称                        | 说明                                            | 默认值 |
| --------------------------- | ----------------------------------------------- | ------ |
| wisenTranslate.origin       | 设置翻译使用的源，可选填，baidu、youdao、google | youdao |
| wisenTranslate.disableHover | 是否禁用鼠标悬浮翻译                            | false  |
